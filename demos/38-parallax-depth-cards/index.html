How It Works

Each card has a --depth CSS variable that controls its position in 3D space. GSAP animates:

  • Transform: scaleZ, translateZ for depth positioning
  • Opacity: Cards closer to viewer are more opaque
  • Blur: Optional blur increases with distance (atmospheric perspective)
  • Delay: Each card stagger is calculated from its depth value

Technique Stack

This uses CSS 3D transforms with perspective container, GSAP for entrance choreography, and ScrollTrigger for scroll-based reveal. Hover states add interactivity.
